# Jorgesay
[Cowsay](https://github.com/tnalpgge/rank-amateur-cowsay) but without cows


## Installation


```ruby
gem 'jorgesay'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install jorgesay

## Usage
```
$ jorgesay hello world
```
